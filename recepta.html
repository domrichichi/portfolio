Recepta

The development around recepta is hinged on UI UX design, as well as some aspects of branding and colour application. Recepta is an economical application that helps users log their purchases and receipts, and making the process of returning items much more seamless.

The project began with finding and researching the target audience. Empathy maps and usability testing provided a stronger understanding on user needs, which resulted in a very linear and user friendly flow.
